Formularios

Se usan para obtener datos e información (input) del usuario para poder procesarlos, mostrarlos o guardarlos segun sea su proposito.

Ejemplo de formularios

input tipo text: Es usado para ingresar textos no tan largos.

Estos input tienen las propiedades siguientes que son las mas basicas:

  • requires: Se usa required para que sea campo obligatorio
  • name: Se usa name para identificar la etiqueta y poderla procesar debidamente
  • minlength: para delimitar los caracteres a entrar
  • placeholder: Se usa placeholder son indicaciones para el usuario
