Una pagina web tiene una estructura digital, la cual se especifica por capas y cada elemento de la página web esta dentro de un "caja" más grande

La página más grande o que contiene las demas cajas es la etiqueta html. Dentro de esta etiqueta se encuenta:

  • head: Contiene los elementos que no son visibles para el usuario.
  • body: Contiene los elementos que son visibles para el usuario.

Dentro de head se encuentra principalmente la etiqueta:

  • title
